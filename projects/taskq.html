TaskQ
Async Task Router

A minimal async task router: users submit a text prompt, GPT detects the task type (summarize, rewrite, explain), and the work is routed to a Redis-backed background worker. A chat-style React UI keeps conversation memory via UUIDs, and the whole stack runs with Docker Compose.

Features

  • Detects task type using GPT, then routes to the appropriate handler.
  • Asynchronous background processing via a Redis queue — the API stays responsive.
  • PostgreSQL stores task history and per-conversation memory keyed by UUID.
  • Chat-like React + Tailwind interface with scrollback and a fixed input bar.
  • Dockerized with Compose for one-command local development.
