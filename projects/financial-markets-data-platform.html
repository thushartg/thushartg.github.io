What it does

Ingestion

Two Python scripts land raw data into BigQuery's raw layer unchanged. Raw data is append-only and immutable — if a downstream transformation breaks, data can be reprocessed without re-hitting the APIs.

  • Yahoo Finance — daily OHLCV prices for SPY, AAPL, MSFT, TSLA, NVDA (1 year of history).
  • FRED API — Federal Funds Rate, CPI, Unemployment Rate, 10Y–2Y yield curve spread, and GDP (from 2020).

Transformation (dbt)

  • Staging — stg_stock_prices, stg_macro_indicators materialize as views: clean column names, cast types, remove nulls.
  • Marts — mart_stock_macro_analysis materializes as a table: daily returns via a LAG window function, 30-day rolling average prices, and macro conditions joined to each trading day.

Data quality & orchestration

  • 7 dbt tests run after every transformation — not-null checks on key columns and accepted-values checks on ticker symbols.
  • An Airflow DAG runs the full ingest → transform → test flow on a weekday cron, fully containerized with Docker.

Output — live dashboard

The mart layer feeds an interactive Looker Studio dashboard with ticker filtering, returns, rolling averages and macro overlays.
